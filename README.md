# Classical Mechanics Physics Informed Neural Networks - CMPINN

A ideia deste repositório é a criação de redes neurais informadas por física (PINN) para problemas de física e mecânica clássica.

O repositório terá 3 pastas principais:

references/ -> repositório onde será salvo, por pasta, as principais referências de artigos e livros a respeito de cada tópico
problems/ -> repositório contendo uma pasta para cada problema enfrentado. Todo o código dos modelos, experimentações, etc, irá dentro de cada pasta de cada problema.
notes/ -> repositório contendo notas e anotações relevantes para cada problema enfrentado, disponível em arquivos .MD onde terá divisor por datas de atualização
